# utils.nvim

A collection of useful utilities for neovim. 

## Installation

Use your preferred Neovim plugin manager to install mru.nvim.

Using [nvim-plug](https://github.com/wsdjeg/nvim-plug)

```lua
require('plug').add({
    { 'wsdjeg/utils.nvim' }
})
```

Then use `:PlugInstall utils.nvim` to install this plugin.

Using [luarocks](https://luarocks.org/)

```
luarocks install utils.nvim
```
